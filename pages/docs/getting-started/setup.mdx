# Setup

Canopy IIIF is built on a [Next.js](https://nextjs.org/docs) platform where production and development builds will follow Next.js documentation.

**System Requirements**

- Node.js 18.17 or later recommended.
- macOS, Windows (including WSL), and Linux are supported.

## Create a Project

### Using a Github template

Note that this step requires a GitHub account.

1.  Navigate to the [Canopy IIIF repository](https://github.com/canopy-iiif/canopy-iiif) in GitHub
2.  Click the _"Use this template"_ button
3.  Click the _"Create a new repository"_ button
4.  Select the account you want to own the repository
5.  Enter a repository name and optional description
6.  Click the _"Create repository"_ button

### Using `git clone`

1. In your terminal, navigate to the directory where you want to create your project
2. Clone the `canopy-iiif/canopy-iiif` repository
3. Navigate to the `canopy-iiif` directory

```bash copy
cd ~/Projects
git clone git@github.com:canopy-iiif/canopy-iiif.git
cd canopy-iiif
```
